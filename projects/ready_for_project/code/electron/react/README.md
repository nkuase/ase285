# Simple React Calculator

A basic calculator app built with React.

## Features
- Two number inputs
- Four operations: Add, Subtract, Multiply, Divide
- Real-time result display

## Installation

```bash
npm install
```

## Run

```bash
npm start
```

Open browser at `http://localhost:5173`

## How It Works

1. **State Management**: Uses `useState` to manage inputs and results
2. **Calculate Function**: Performs operations based on selected operator
3. **Input Validation**: Checks for valid numbers and division by zero
